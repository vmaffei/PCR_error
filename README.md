## pcrror.py
Multithreaded PCR error simulator

### Arguments
* '-i'
  * The fasta to PCR
* '-o'
  * The output file
* '-c'
  * The number of PCR cycles to simulate
* '-e'
  * The inverse error rate
* '-s'
  * The seed for the Psuedo-Random number generator
* '-t'
  * The number of threads to start
